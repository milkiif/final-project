# Web-Based Human Resource Management System (HRMS)

## 📌 Project Description
This is a web-based HRMS developed as a final year project. 
The system helps manage employees, leave requests, vacancy announcements, and work schedule preparation.

## 🚀 Features
- Employee Management
- Leave Management
- Vacancy Management
- Work Schedule Preparation
- Admin Dashboard

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
